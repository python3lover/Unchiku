「うんちく」は日本語ベースのプログラミング言語です。英語をまだ習っていない幼稚園生や小学生でも、普段使っている日本語でプログラムが書けます。うんちくでのプログラミングは、コンピューターにしてほしいことを手紙に書いてお願いするような感じです。 大阪弁にも対応してます。カナダで育った日本人の僕が、たくさんの人に気軽にプログラミングを楽しんでもらいたいと思って作りました。ぜひサンプルファイルのデモを見たり、実行ファイルをダウンロードして遊んでみてください。

⚠️　下の再生ボタン▶️を押してください。うんちくのデモが始まります。デモ用のサンプルファイルは4種類あり、サンプルファイル２の実行方法を下記します。他のファイルは、数字の2の部分を、1または3、4に置き換えて実行してください。

  1. 上部にある再生ボタン▶️をクリックする。
  2. 説明文の最後に? Fileが表示されたら、Sample 2.unchk と入力し、キーボードのEnterキーを押してください。
  3. Sample 2は林檎の合計金額を計算するプログラムです。最初に林檎1個の値段がコンピューターから質問されるので、数字を入力してください。例えば、100円の場合は、100と入力後、キーボードのEnterキーを押してください。
  4. 次に林檎の個数がコンピューターから質問されるので、数字を入力してください。例えば、10個の場合は、10と入力後、キーボードのEnterキーを押してください。
  5. コンピューターが合計金額を教えてくれ、Sample 2のデモが終了します。

    誰でも簡単にテキストベースのプログラミングを楽しんでほしい、その思って「うんちく」を作りました。下のスライドと動画は、v0.3.0を「Tech Kids Grand Prix」の決勝プレゼンテーション（2018年9月）で発表した際のものです。
